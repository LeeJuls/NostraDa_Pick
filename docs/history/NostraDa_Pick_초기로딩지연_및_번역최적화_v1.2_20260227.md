# [GA] 히스토리 기록: 초기 로딩 지연 및 무한 로딩 버그 수정

## 1. 개요
- **일자**: 2026-02-27
- **버전**: v1.2
- **작성자**: Antigravity (GA)
- **주요 내용**: 홈 진입 시 데이터가 즉시 로드되지 않는 현상(무한 로딩) 해결 및 데이터 로딩 속도 최적화

## 2. 의사 결정 및 논의 사항
- **병렬 처리 도입**: 기존의 이슈 리스트 렌더링 시 루프 내부에서 순차적으로 `await` 하던 번역 호출을 `Promise.all`을 이용한 병렬 처리로 변경함. 이를 통해 이슈 개수가 많아져도 로딩 시간이 일정하게 유지되도록 최적화함.
- **방어적 프로그래밍**: API 호출 실패 시 "문제를 불러오는 중..."에 멈춰있지 않고 에러 메시지를 표시하거나 초기화하도록 예외 처리를 강화함.
- **UI 레이아웃 보완**: `index.html`에 `my-recent-correct` 컨테이너를 명시적으로 추가하여 통계 로드 시의 DOM 조작 안정성을 높임.

## 3. 변경 상세 (Code Changes)
- **Backend**: (변경 없음)
- **Frontend**:
  - `static/js/app.js`: 
    - `loadIssues`, `loadLeaderboard`, `loadMyStats` 함수에 try-catch 및 에러 핸들링 추가.
    - 이슈 제목/카테고리 번역 시 병렬 처리(`Promise.all`) 적용.
  - `templates/index.html`: `my-recent-correct` 디비전 추가.
