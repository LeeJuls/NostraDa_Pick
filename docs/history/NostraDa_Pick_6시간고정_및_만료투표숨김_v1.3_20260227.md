# [GA] 히스토리 기록: 투표 마감 6시간 고정 및 만료된 투표 숨김 처리

## 1. 개요
- **일자**: 2026-02-27
- **버전**: v1.3
- **작성자**: Antigravity (GA)
- **주요 내용**: 모든 퀴즈 투표 시간을 6시간으로 강제 고정하고, 마감 후 24시간이 경과한 투표는 화면에서 제거.

## 2. 의사 결정 및 논의 사항
- **마감 시간 고정**: 사용자 요청에 따라 질문 생성 시 Gemini가 임의로 설정하던 투표 기간(1~6시간)을 완전히 무시하고, 모든 기존 및 신규 이슈의 마감 시간을 **"생성 시간 + 6시간"**으로 일괄 고정함. 이를 위해 `update_deadlines.py`를 일회성 스크립트로 실행하여 기존 DB 레코드를 업데이트 완료함.
- **오래된 투표 숨김(Retention) 기준 변경**: 마감 시간(`close_at`) 기준이 아니라, **"실제로 답변이 확정(RESOLVED)된 시간(`resolved_at`)"**을 기준으로 24시간이 경과한 이슈만 숨기도록 로직을 정정함. 답변 처리가 지연되어 질문 마감만 된 상태(OPEN)의 이슈는 계속 화면에 노출되도록 보장함.
## 3. 변경 상세 (Code Changes)
- **Backend**:
  - `services/gemini_service.py`: `save_issues_to_db` 내부 구문을 수정하여 마감 시간을 무조건 `생성시간 + 6시간`으로 변경.
  - `routes/api.py`: `get_open_issues` 쿼리에 `gte('close_at', hide_threshold)` 조건 추가.
  - `update_deadlines.py`: [NEW] 일회성 마이그레이션 스크립트. 기존 데이터 4건 수정 완료.

## 4. 커밋 메시지
- `[GA] 투표 지속 시간 6시간 고정 및 24시간 지난 마감 투표 숨김 로직 추가`
