📏 STRICT RULES (문제 품질 가드레일)

ARTICLE-BASED: 지정 기사 주제만
ABSOLUTE UTC TIME: 절대 시각 표기
FUTURE ONLY: 미래 이벤트만
RESOLVABLE: 마감 전 판정 가능
WITHIN 48H: 48시간 이내
PRICE/MARKET: 24시간 이내 + ±5%
SPORTS: API 일정만 사용
SPORTS SCORES: 정수만
ENTERTAINMENT: 출시/발표 금지
SPECIFIC ENTITIES: 구체적 고유명사
ANSWER NOT KNOWN: 답 알려진 문제 금지
MINIMUM DEADLINE: 뉴스 24h 이상
SPORTS — NO DRAW: "draw" 동사 금지 → "finish tied"
ECONOMY DATA: 발표된 GDP/CPI → 시장반응 질문
NAMED ENTITY: "any country/official" 금지
POLITICS PATTERNS: Poll/Legal/Vote/Named action만
TECH KEYNOTE: mention/discuss 금지 → binary outcome

BANNED STATEMENTS: follow-up 성명·모호한 제재 금지
CATEGORY: 주가→economy, 분산 max_per_cat
